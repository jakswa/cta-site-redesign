Redesigning the Civic Tech Atlanta website to better serve volunteers, partners, and the public. The new site features a project showcase, first-timer guide, partner pathway, and blog for community updates.

## Goals

- Make it easy for first-timers to understand how to get involved
- Showcase active and past projects to demonstrate impact
- Create a clear pathway for potential partner organizations
- Improve mobile experience and accessibility

## Get Involved

This project is actively seeking contributors! We need help with:

- Front-end development (HTML, CSS, Jekyll/Liquid)
- Content writing and editing
- UX feedback and testing
- Accessibility review
