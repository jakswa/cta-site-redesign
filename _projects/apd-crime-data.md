Analysis and visualization of publicly available crime data from the Atlanta Police Department. This project worked with APD's Crime Data Downloads to make the information more accessible and useful for researchers and community members.

## Project Goals

- Parse and clean APD crime data exports
- Create visualizations of crime patterns
- Make public safety data more accessible
- Support data-informed community discussions

## Status

This project is archived. The analysis tools and methods remain available for reference.
